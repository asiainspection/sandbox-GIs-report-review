# Hallmark (HMK) — GI Rules Reference (Report-Content Scope)

Rules below reflect only what must be checkable in the written inspection report (checkpoint answers, photos, comments/remarks, values entered, consistency between fields). Rules that were contradicted by an actual passed Hallmark report (1HGO3545, PSI, 12-Jun-2026) have been removed or corrected to match the real checklist wording.

> ⚠️ TO CONFIRM: The product-type table says "Non-HA/Non-HCLP 2D items → Double sampling," but the sampling-matrix section says the choice depends on packing method: mixed-packed → Single, individually packed → Double. Apply the more specific rule but confirm with back office (SMP.5.1 / SMP.5.27).

| ID | Field/Location | What to check | Severity |

Normally only KC (Kansas City) orders carry an HA/Non-HA classification; if the BU/client does not classify the item, select "N/A." | BLOCKING if misidentified where a classification exists |
| HA.2.2 | Highly Confidential Licensing Product (HCLP) checkpoint | Same logic as HA: determined per specs; "Yes"/"No" doesn't itself fail the result; if BU/client doesn't classify, select "N/A." | BLOCKING if misidentified where a classification exists |
| HA.2.3 | Specific Checkpoints for HA & HCLP (when applicable) | Check "Products Integrity Notes" in spec; adopt Single sampling plan for HA item; ask factory for PP report; contact Hallmark Engineer for 1st-lot inspections. For HCLP: do NOT show the entire product picture — use a plank picture and remark "HCLP, no photo"; defect photos must focus only on the defect area. | BLOCKING when HA/HCLP applies and not followed |
| HA.2.4 | HMK Inspection Reason field | Select the reason showing why Hallmark inspection is required (per IRF/supplier verification) — for data-analytics purposes; supplier-internal inspection is not applicable. If "CQS/SEP - Retailer/licensor request - Other (cmnt)" is selected, the retailer/licensor name must be specified in Comments. If "CQS/SEP - Other reason (must get HMK mgr approval)" is selected, this signals the inspection is out of Hallmark's normal scope and requires prior Hallmark Quality Manager approval, with the reason specified. | BLOCKING if reason unspecified/unsupported |
| HA.2.5 | Business Unit field | The appropriate BU must be selected based on the responsible procurement team / client destination (e.g., HO, KPS, KC Direct, Crayola, Gifts, Greetings, International, Print Specialty). | MINOR if wrong BU selected (affects routing, not product conformity) |
| HA.2.6 | Client/Destination field | Must match the actual destination/program (e.g., "KC HKBO" for Hallmark-brand US orders routed via Hong Kong back office). | MINOR |
| HA.2.7 | SAP Group Number field | Must match the number shown on the spec for this SKU; this number determines the applicable AOQL%/inspection type. | BLOCKING if mismatched |

| ID | Field/Location | What to check | Severity |
|---|---|---|---|
| CS.3.1 | Carton Range checkpoint | Report must state the actual range of cartons inspected (e.g., "1~7"). | MINOR if missing/inconsistent with lot |
| CS.3.2 | Shipper Dimensions & Weight & Marks | Report must record measured Length x Width x Height and check ship marks against spec. Hallmark general requirement: Min O.D. L12"xW8"xD5" (305x203x127mm); Max O.D. L22"xW21"xD32" (559x533x813mm); weight 5–50 lbs (50 lbs preferred as an upper ceiling, ideally under 35 lbs). Dayspring/Mary & Martha: max 24"x24"x14.5" (609x609x368mm), max 30 lbs. Keepsakes AU/UK: per their specific packaging guideline attachments. Walgreens FOB: max L48"xW40"xH/D46" (121.9x101.6x116.8cm), 50 lbs, no minimum; identifiable via "Inspiration from Hallmark" logo + WIC# on the carton. | BLOCKING if outside range and unremarked |
| CS.3.3 | Carton Labels & UPC Check | Content on the label must match PO & Specification (SKU#, barcode, item number, SAP material number, wholesale quantity, etc.); UPC code must correctly match the SKU/material number; UPC print quality (ink density, line width) must meet ANSI/UPC Council Symbol spec. | BLOCKING |
| CS.3.4 | Partially Filled Carton (when applicable) | Check: partially-filled label present per manual; quantity & assortment inside carton match the corner label; barcode is scannable; quantity on corner label is printed (not handwritten). | BLOCKING if any point fails |
| CS.3.5 | Shipper Packaging | Check (18 cartons): shipper quality; wholesale quantity in shipper consistent with specs, ship test report, and carton (corner) label; carton quality (5-ply/5-layer). | BLOCKING |
| CS.3.6 | Wholesale Packaging | Check (18 wholesale packages): wholesale label content; retail units packed per wholesale; orientation and packaging material; consistency with Specs, PO, ship test report, and commit sheet. If shipped through a Hallmark DC then onward: must follow the new wholesale labeling guidelines (UPC wholesale label + scan-ability check). If shipped directly to a customer (not via Hallmark DC): previous labeling practice applies. | BLOCKING |
| CS.3.7 | Retail Qty per Wholesale | Recorded retail quantity per wholesale must conform to Specs, PO, ship test report, and (for HO) commit sheet. | BLOCKING if mismatched |
| CS.3.8 | Retail Packaging | Match against DAS; check PO/Spec/ship test report for: retail quantity and package; packing component & sequence; pull more samples if any issue found; ceramic tableware care instructions consistency; HO — product/colorway/UPC/package/price/copyright/photo per spec and commit sheet; package instructions vs. actual function; battery packed separately (Keepsake); CARB label; spelling. | BLOCKING (quantity/spec/labeling mismatch); MINOR (spelling) |
| CS.3.9 | Retail Labeling, UPC & COO | Label/logo placement and content on retail package and product must be checked and discrepancies remarked; product labels must be removable for verification but strong enough not to fall off; Country of Origin correct; UPC validated and scannable; price correct if applicable. | BLOCKING |
| CS.3.10 | Warning Statement and License | Warning statements and licensing info must be verified per PLM specs, where applicable. | BLOCKING if non-conforming |
| CS.3.11 | Customer Special Package Requirement (when applicable) | Retailer-specific packaging rules apply where relevant (e.g., Amazon: 1 pc retail per wholesale, shippable wholesale with special label, HMK corner label on shipper, ISTA 3A test profile; Dayspring: label on both product and white box; Mary & Martha: label on white box only). | BLOCKING when the named retailer/program applies and is not followed |
| CS.3.12 | Polybag (when applicable) | Compare against the approved sample: material (PP or PE); thickness >1 mil (0.0254mm) except Toys (refer to SR/FR report); vent hole present; warning content in English, French, and Spanish. | BLOCKING |

## 6. Tests (results/values that must appear in the report)

| ID | Field/Location | What to check | Severity |
|---|---|---|---|
| TST.6.1 | Function Test (1 cycle, full sample) | Product must perform its intended function properly for 1 cycle (electronic, mechanical, assembly, weighting, usage, writing, etc.). Applies only to products with a functional attribute intended for consumer use. Fitting Test for cards with envelopes: 5 pcs, insert smoothly. | BLOCKING |
| TST.6.2 | Function Test (25 cycles) | Product must perform its intended function properly 25 times. Not applicable to one-time-use products (candles, paper bags, paper plates, paper cups, napkins, etc. with no functional component) — mark N/A for these. | BLOCKING if applicable and not tested |
| TST.6.3 | Battery Powered Product Evaluation (electrical-function items) | 5 samples: Operation current, Standby current, Voltage (V), dB. Operation current, Voltage, and dB should conform to spec and be recorded per the FR report. Standby current must be <10µA per FR report. A picture of each test report/reading must be included. | BLOCKING if values not conforming to spec/FR or missing |
| TST.6.4 | Adhesion Tape Test | Applies to non-paper secondary-use attachments on greeting cards, made of plastic or metal material. Test adhesion of coating by applying tape (3M #810) to the painted/coated surface and removing rapidly — coating/paint should not detach from the substrate. Does not apply to fabric and/or plush items. | BLOCKING |
| TST.6.5 | Glue Joint Test | Glue joints must be separated to verify bonding strength/sufficiency; glue joint surface must not have paint, antiquing, or other surface coatings; pegs must be correct size and fit properly. | BLOCKING |
| TST.6.6 | Pull Test | Assembly joints holding the ornament together must withstand a 5 lb pull force; fragile/decorative attachments must withstand a 2 lb pull force. Pull strength must be recorded with a comment. | BLOCKING |
| TST.6.7 | Carton Drop Test | Required in the 1st lot if the product meets one of: special request from HMK ENG in spec; ceramic & glass product; a product dimension exceeds 16"; the item failed ship test one or more times. Fill 65%–75% of the shipper carton with wholesale boxes, follow transport environment test procedure #1301, and perform 14 drops. Sample: minimum 12 retail units, maximum 3 shipping cartons. Not required again in sequence lots if the 1st lot passed. | BLOCKING when triggered and not performed/recorded |
| TST.6.8 | Hanger Test (products with a hanger/hook/cutout) | Sample suspended from the packaging hanger as intended; tension force equal to 3× the packaged product weight applied gradually and held 1 minute, pulling downward. | BLOCKING |
| TST.6.9 | Ceramic Tableware (when applicable) | Check supplier's internal acidic/dishwasher/microwave/high-temp-pressure test records; use a cutter on decorated areas to check decal firing quality (no obvious scratch); Hot & Cold Water Test on 3 samples (3 cycles: room temp → 90-100°C water 30s → 0-4°C water 30s → room temp 5 min → check for cracks/damage). | BLOCKING |
| TST.6.10 | Fabric and Plush Item (when applicable) | No bleeding risk — verify a high-contrast fabric pair passed the bleed test; stuffing must be new and clean. | BLOCKING |
| TST.6.11 | Paper/plastic labeling or lamination on product (when applicable) | Laminated layer must be secured and permanently adhered; discuss with PE if it's easily removable. | BLOCKING if lamination easily removable and unescalated |
| TST.6.12 | Moisture Test (natural material, when applicable) | Moisture should be 8–12%; if higher, sufficient desiccant must be packed to avoid mold risk. | BLOCKING |
| TST.6.13 | Sharp Point & Sharp Edge Check | Full sample checked for sharp points/edges. | BLOCKING |
| TST.6.14 | Safety Test (Toys only, when applicable) | Pull/torque/drop tests with acceptable results; for age 3 and under: pull 15+5 lbs (21+5 lbs for EU) per HQR, no small parts; no sharp point/edge for age 8 and under (as received and after abuse). Plush items additionally require stuffing cleanliness (sample 1) and metal detection (sample 10) after the safety test. | BLOCKING |
| TST.6.15 | Ventilated Holes (art closures, e.g. marker caps, when applicable) | Holes must be completely open. | BLOCKING |
| TST.6.16 | Odor/Smell Check | Full sample checked after opening all packaging; no offensive odor (musty, petroleum, strong pungent paint, or any unintended aroma) acceptable on product or packaging. | BLOCKING |
| TST.6.17 | Additional Tests (when applicable) | Any additional test not otherwise listed (e.g., Tearing Test for glued components/books/multi-page cards/paper boxes — sample 1; 6" Table Drop Test if spec requires — sample 1; Incline Test if spec requires — sample 1) must be named in the comment field with its result recorded. | MINOR (missing test name in comment) / BLOCKING (test omitted when spec requires it) |
| TST.6.18 | Remark for Report Modification | Applicable only when modifying an already-submitted report — not applicable on first report submission (correctly N/A on initial reports). | N/A on 1st submission is not an error |

## 10. Do-not-flag — client-specific notes

- Minor print defects under 5mm on felt ornaments, unless they affect sellability.
- Dirt marks on kraft recycled paper products, provided no glue residue has penetrated the bag surface.
- A "Minor" defect recorded under a standard AOQL plan where Minor Acceptance = 0 is itself an error, not a do-not-flag case (see SMP.4.29) — except for plush items.
- N/A on Reece's Law checkpoint for items without button-cell battery or non-US/Canada destination.
- N/A on HA/HCLP checkpoints when the BU/client does not classify the item.
- N/A on PID Number checkpoint for 3D products or other non-2D-KC-card products.
- N/A on Plastic Free Policy Check for BU/clients other than HMK UK, M&S, or Lego.
- N/A on Battery Powered Product Evaluation, Ceramic Tableware, Fabric/Plush, Moisture Test, Safety Test (Toys), Ventilated Holes, Carton Drop Test, Customer Special Package Requirement, and Polybag checkpoints when the product genuinely does not fall into the applicable category.
- "Remark for Report Modification" correctly showing N/A on a first-time (non-modified) report submission.
